Install missing CODECs for HTML5

You may notice issues or tasks in the System Overview of the dashboard. One of the items is "" with the following description: "You are missing support for the following HTML5 codecs: m4a. To fully support HTML5 browser playback you will need to install programs that can not be distributed with FreePBX. If you'd like to install the binaries needed for these conversions click 'Resolve' in the lower left corner of this message. You can also safely ignore this message but browser playback might not work in your browser." In order to resolve and understand this item, you may click the resolve button that will take you to a page on freepbx.org that will provide more information.

  • SOX is already installed on my installation but you may attempt to install it by running the following command:
    apt-get install sox
  • MPG321 may be installed by running the following command:
    apt-get install mpg321
  • LAME is already installed on my installation but you may attempt to install it by running the following command:
    apt-get install lame
  • FFMPEG may be installed. A good resource for installing on Raspberry Pi may be found at https://lindevs.com/install-ffmpeg-on-raspberry-pi/
    You may run the following commands to install FFMPEG:
    sudo apt update
    sudo apt install -y ffmpeg
