Global Preferences (help?)
Enable Phone Security:
Phone security prevents unregistered phones from viewing the directory. If phone security is disabled, all unregistered phones will have access to objects marked 'Public'.
Enable Object Security:
Object security allows restriction of phones to either 'Public' or 'Private' data based on each phones security level.
Enable Prefix: Prefix:
Enabling the prefix will automatically attach a prefix to the front of all contact numbers.
Order Memos By:
Choose how to order the memos posted to the phone.
