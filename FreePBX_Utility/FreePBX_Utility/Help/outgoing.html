FreePBX Utility Help - Outgoing Calls

The Outgoing Calls tab displays a report of calls routed through the OBi110 ATA to external destinations. If you are using another device, you may need to create a new report PHP page or modify the existing query. The following are the files of the GitHub project associated with the Outgoing Calls report.

  • RasPBX_Build/Quick Start/www/html/utility/outbound.php
  • RasPBX_Build/Quick Start/www/html/utility/queries/outbound.txt
