FreePBX Utility Help - Inbound Calls

The Inbound Calls tab displays a report of external calls received through the OBi110 ATA. If you are using another device, you may need to create a new report PHP page or modify the existing query. The following are the files of the GitHub project associated with the Inbound report.

  • RasPBX_Build/Quick Start/www/html/utility/inbound.php
  • RasPBX_Build/Quick Start/www/html/utility/queries/inbound.txt
